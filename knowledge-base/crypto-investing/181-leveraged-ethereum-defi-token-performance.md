## 槓桿版以太坊的代幣選擇策略

關於槓桿版以太坊的話題：

Layer 2 代幣似乎不是最佳選擇。從歷史數據來看，大型 DeFi 類代幣表現更好，例如：

- **ENS** (Ethereum Name Service)
- **UNI** (Uniswap)
- **LDO** (Lido，或 Eigen)
- **ENA** (Ethena)
- **Pendle**

（以上並非全部概括,僅為舉例說明）
