## 核心策略

將投資決策委託給不同賽道的專業 KOL，實現專業化分工：

### 賽道與對應 KOL

| 賽道 | 委託對象 |
|------|----------|
| **Solana** | @Michael_Liu93、@0xcryptowizard |
| **Base** | @superbili、@hellosuoha |
| **空投挖礦** | @hebi555、@MetaHunter168 |
| **礦業與股票** | @Vito_168 |
| **一級市場** | @BlueV1007、@Lin_2mars |
| **BNB Chain** | 自己研究 |

### 策略說明

- 每個賽道跟隨 1-2 位專業 KOL
- 將該賽道的決策權完全委託給對應的 KOL
- 僅保留 BNB Chain 賽道的自主判斷
