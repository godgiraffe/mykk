# 賣出看跌期權抄底策略（Cash Secured Put）

> **日期**: 2026-02-05
> **標籤**: `期權` `Cash Secured Put` `抄底` `BTC` `ETH`

---

## 策略概述

心理價位是 $65,000 要建 BTC 長期倉位時：
1. **賣出** BTC $65,000 的看跌期權
2. **同時**現價掛單 $65,000 的 BTC 買單

## 兩種結果

| 情境 | 結果 |
|------|------|
| 到期日 BTC 跌破 $65,000 | 抄底成功 + 賺取賣出期權的權利金 |
| 到期日 BTC 未跌破 $65,000 | 抄底失敗，但賺取權利金 |

## 實際範例

| 參數 | 數值 |
|------|------|
| 本金 | 1,000 U |
| 期權類型 | 賣出看跌期權（Sell Put） |
| 到期日 | 2026-02-27 |
| 行權價 | $65,000 |
| 收取權利金 | 150 U |
| 保證金餘額 | 立即變為 1,150 U |

**收益率**：不到 1 個月 **15%** 的收益

## 熊市末期的複利效應

- 可以持續掛這個期權，直到抄底成功
- 每次到期未觸及 → 收取權利金
- 最終抄底成功時，實際成本比直接抄的人**低非常多**

> 熊市末期，你既抄到了底，又讓賭短期波動的人支付了你的抄底本金。

## 重要注意事項

- Binance 的期權是**歐式期權**，Cash Settled
- 必須**同時掛現貨買單**才能做成真正的 Cash Secured Put
- 使用新交易工具前一定先自己研究明白
