## 推薦技術棧

我推薦給新手獨立開發者的技術棧如下：

**前端**：
- React
- React Router V7（替代 NextJS）
- TailwindCSS

**後端**：
- Bun + Hono，或者
- Python + FastAPI

**資料庫**：
- MySQL/PostgreSQL
- Redis

**CI/CD**：
- GitHub Actions
- pm2
- supervisor
