雷電模擬器多帳號操作及獨立 IP 設置

啟動模擬器，不單是為了 TON 鏈的小遊戲，而是我覺得模擬器作為擼毛的一項工具，或許以後要不可或缺了。如果是模擬真實手機，我覺得獨立 IP 也挺有必要。

本教程主要為怎麼設置多帳號及獨立 IP 設置。

## 一、雷電模擬器下載與基礎設置

雷電模擬器下載，下載雷電 9 安卓 9.0（64 位）最好下載一個空間大點的盤，後期使用很佔內存。

**下載與安裝**

1. 下載鏈接：https://www.ldmnq.com/

2. 下載安裝以後桌面會有兩個圖標，打開雷電多開器進行設置

**基礎窗口設置**

3. 選擇默認的第一個雷電模擬器進行設置（設置好以後此窗口以後僅用於添加複製模擬器之用，不要用他來作為帳號使用）

   點擊三點選擇設置，跳轉窗口 → 性能設置 → 手機版 → 720x1280，最後保存設置

4. 然後再到其他設置將 root 權限打開，開啟本地連接，最後保存設置

**批量複製窗口**

5. 點擊右下角添加模擬器，選擇複製模擬器，跳轉小頁面，複製對象一定要選擇剛剛自己設置的以後不使用的那個窗口，複製數量最多 10，最後點擊複製

   第一個作為複製對象不讓用是怕登錄了電報號被複製的時候對沖掉線

**窗口排列**

6. 點擊左下角梅花標誌對窗口排列進行設置，我選了宮格排列，每行 10 個，你們可根據自身需求設置

7. 勾選雷電窗口，選擇批量操作 → 啟動窗口，窗口啟動後點擊左下角四方格，窗口會自動排列好

## 二、下載電報及梯子的 APK 拖進模擬器裡面

**獲取 APK 文件**

1. 這倆 APK 我沒法上傳到文章裡，你們可自行搜索下載，實在沒有去我電報群或者私信我，我發你們（我自己在用，但我不是開發技術人員，安全性不保證）

   電報群：https://t.me/+-BAMp2dvPG45Y2Vl

**Postern 代理設置**

2. Postern 拖進模擬器裡面點擊下載，顯示版本報錯不要緊的，可以使用

   打開 Postern 選擇添加代理服務器，IP 商給我們的 IP 段位一般都是四段，按我標記的複製填寫進去即可，自己添加服務器名稱，代理類型選擇 SOCKS5，然後保存

**配置規則**

3. IP 保存好以後點擊左上角四條槓，選擇配置規則 → 點開規則 1，將動作修改為通過代理連接，代理選擇剛保存的 IP，然後下拉保存

4. 返回選擇規則 2，將動作修改為通過代理連接，代理選擇剛剛自己保存的 IP，然後再保存，保存完再單擊左上角四條槓，將 VPN 關閉，然後再打開

**驗證 IP**

5. 返回主頁面，系統應用 → 瀏覽器 → 百度搜索輸入【IP】，查詢自己的 IP 歸屬地，如果是自己綁定的 IP 即成功

## 三、登錄電報號與多號操作

登錄電報號，多號操作，同步器使用選擇更多 → 同步器 → 勾選窗口同步操作
