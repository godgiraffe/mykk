## 問題背景

X 的官方 API 價格過高，即便是第三方 API 也要 20 多美元一個月。

## 解決方案：Twikit

Twikit 是一個基於 Python 的開源 X API 替代方案。

**運作原理**：
- 登入一個 X 帳號
- 模擬瀏覽器訪問 https://t.co/JBKJXNyB9i
- 透過自動化方式取得 X 平台的資料

**主要優勢**：
- 完全免費
- 開源專案
- 基於 Python，易於整合到現有專案
