小草幣價猛拉，DePIN 熱又來了。以下是一些能掛機並且本人正在掛的項目：

## 推薦項目

### Gradient Network (@Gradient_HQ)
這個項目對 IP 要求很高，目前還沒找到低價的 IP。

### Dawn Internet (@dawninternet)
- 已發布腳本
- 對 IP 要求很低，單號單 IP 即可
- 多 IP 收益不疊加

### Blockmesh (@blockmesh_xyz)
這個我用的是華哥 @ferdie_jhovie 的一鍵腳本，大夥感興趣的可以看一下。
