## 詳細介紹

### 1. DotSwap Tools
- **開發團隊**: @dot_swap
- **網址**: https://tools.dotswap.app
- **服務費**: 每張 0.01 FB
- **批量鑄造**: 支援，最多 25 張
- **特色**:
  - 目前最強的 CAT20 鑄造工具
  - 速度很快
  - 鑄造失敗可以退款

### 2. CAT20 瀏覽器
- **開發團隊**: @zen913
- **網址**: https://cat20.zenwav.com
- **服務費**: 每張 0.01 FB
- **批量鑄造**: 不支援
- **特色**: 非常好用的 CAT20 瀏覽器

### 3. IDClub
- **開發團隊**: @idclub_ord
- **網址**: https://fractal.idclub.io
- **服務費**: 按比例收費，每張超過 0.05 FB（較貴）
- **批量鑄造**: 支援，最多 3 張
- **特色**: 曾經很火的 BRC20 鑄造工具

### 4. CoinTool
- **開發團隊**: @cointool
- **網址**: https://cointool.app/fractal/cat20/mint
- **服務費**: 0.005 FB
- **批量鑄造**: 支援，可以設定批量鑄造張數
- **特色**:
  - 最早支援 CAT20 的工具之一
  - 需要上傳助記詞
  - 需要排隊

### 5. Gaze Network
- **開發團隊**: @Gaze_Network
- **網址**: https://gaze.network/fractal/cat20/mint
- **服務費**: 每張 0.05 FB
- **批量鑄造**: 不支援

### 6. CAT Protocol Tracker
- **開發團隊**: @hbeckeri
- **網址**: https://catprotocol.org
- **特色**: 作者測試多次都打不上，建議多試幾次
