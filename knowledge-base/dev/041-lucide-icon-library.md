## 簡介

Lucide 是一個開源的 SVG 圖示庫，提供超過 5000 個設計精美的圖示。在 Vibe Coding 的開發過程中發現這個圖示庫非常好用，強烈推薦給前端開發者使用。

官方網站：https://lucide.dev/icons/

## 自定義功能

Lucide 提供圖示自定義選項：

- **Color**（顏色）：自訂圖示顏色
- **Stroke width**（筆劃寬度）：預設 2px，可調整
- **Size**（尺寸）：預設 24px，可調整
- **Absolute stroke width**（絕對筆劃寬度）：固定筆劃寬度選項
- **Include external libs**（包含外部函式庫）：整合第三方工具
