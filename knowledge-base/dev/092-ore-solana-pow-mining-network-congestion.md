## 問題背景

你有發現你的 Solana 整個卡鏈了嗎？主要是因為大家都在打 @OreSupply，這是 Solana 上的 POW（工作量證明）挖礦項目。由於 Solana 的創始人轉推造成 FOMO，幾乎把整個 RPC 都打掛了。

## 解決方案：使用私人節點

建議大家網路申請一些私人的免費節點或是自行架設節點：

- **Helius**: https://www.helius.dev/
- **QuickNode**: https://www.quicknode.com/

## 關於 Ore

Ore 是一個全新的可挖礦代幣，於今天在 Solana 區塊鏈上推出。它使用了一種新穎的工作量證明算法，保證休閒礦工永遠不會被排擠出獲得獎勵的機會。這是一種可以從任何地方挖掘的數字貨幣，無論是在家中還是在手機上。

## Helius RPC 服務特點

由於原文包含 Helius 的完整頁面內容，以下整理其核心服務：

### LaserStream（數據流服務）
- 最快、最可靠的 Solana 區塊、交易和帳戶數據流（通過 gRPC）
- 極端冗餘 — 具有歷史重放和持久性的冗餘節點集群
- 優化的 ShredStream — 在不需要運行專用節點的情況下比任何人更早接收數據
- 全球覆蓋 — 區域端點包括 FRA、AMS、TYO、SG、LAX、LON、EWR、PITT 和 SLC

### Sender（交易發送服務）
- 即使在最嚴峻的市場條件下也能可靠地發送交易
- 超低延遲交易發送服務
- 無需 API 積分
- 7 個全球端點
- 針對交易者和 MEV 搜索者優化
- 可並行向 Jito 和 Helius 發送交易
- 通過 7 個區域端點提交交易
- 訪問來自 Solana #1 驗證者的質押帶寬

### Solana RPCs
- 通過全球分佈式 RPC 和 Gatekeeper 邊緣網關實現最快的 Solana 連接路徑
- 通過基於 Rust 的邊緣網關 Gatekeeper 實現亞毫秒級響應
- Helius 專屬的歸檔方法，具有基於游標的分頁功能（如 getTransactionsForAddress）
- 默認通過質押連接發送交易
