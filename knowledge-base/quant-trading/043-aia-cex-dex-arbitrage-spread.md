# AIA 上市套利：流動性解放的價差收斂機會

## 核心觀點

AIA 開放 CEX 提幣後，DEX 與 CEX 之間被阻隔的流動性瞬間打通，產生高達 8-9% 的正價差套利機會。作者在價差約 6% 時果斷進場搬磚，成功把握流動性回流的紅利。

## 套利機會拆解

| 關鍵要素 | 說明 |
|----------|------|
| **觸發事件** | AIA 開放 CEX 提幣功能 |
| **流動性變化** | DEX-CEX 流動性瞬間打通，原本被阻隔的資金開始自由流動 |
| **最高價差** | 8-9% 正價差（DEX 價格高於 CEX） |
| **進場點位** | 6% 價差 |
| **收益結果** | 成功達標，抓住流動性回流紅利 |

## 套利邏輯

### 流動性阻隔 → 價差擴大

當 CEX 尚未開放提幣時：
- DEX 流動性相對獨立
- CEX 資金無法套利至 DEX
- 供需失衡導致 DEX 價格溢價

### 提幣開放 → 價差收斂

一旦 CEX 開放提幣：
- 套利資金可自由流動
- 低買高賣力量進場
- 價差快速收斂

## 實戰要點

1. **時機判斷**：密切關注新幣 CEX 提幣開放時間
2. **果斷執行**：價差足夠大時（如 6%+）立即進場
3. **速度優勢**：流動性釋放後價差收斂很快，需要快速反應
4. **風險控制**：注意網絡擁堵可能導致的交易延遲

## 相關概念

**價差套利（Arbitrage）**：利用同一資產在不同市場的價格差異，低買高賣獲利

**流動性（Liquidity）**：資產能夠快速買賣而不大幅影響價格的能力

**DEX-CEX 套利**：在去中心化交易所與中心化交易所之間進行的套利交易
