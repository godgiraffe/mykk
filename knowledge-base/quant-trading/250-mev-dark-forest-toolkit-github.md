## 專案簡介

GitHub 上的 MEV 工具包倉庫：[dark-forest-toolkit](https://github.com/cypherpunk-symposium/dark-forest-toolkit)

**MEV（Maximal Extractable Value，最大可提取價值）** 涵蓋區塊生產供應鏈中參與方產生的利潤。對於搜尋者（searchers）來說，這些收益可以來自於：

- 跨市場套利（減少市場間的價差）
- 交換用戶的滑點損失
- 搶先交易用戶的交易
- 其他策略

MEV 提取可以成為一種善的力量——在非掠奪性和公平的區塊空間自由市場中，作為經濟安全的激勵機制。供應鏈中的每個參與者都很重要。

**訂單流毒性（Order Flow Toxicity）** 是交易者對擁有私人信息優勢的對手方的暴露程度。如同生活中的一切，**最有價值的商品是信息**。
