## 一、套利或者說價差空間如何出現?

毫無疑問就是價格的變化,價格變化的誘因其實就兩種——事件驅動和數據驅動。

剛好有兩個近期例子:

- **事件驅動型價格變化**: 前幾天 $FDUSD 暴跌就是事件驅動型價格變化,Sun 發文指控 FDT 這一具體事件或者消息導致 FDUSD 遭拋售導致的下跌。

- **數據驅動型價格變化**: $OM 的暴跌就很明顯屬於數據驅動,暴跌但是短時間沒有一個具體的消息面,純粹是資金異動導致的。

我們知道,快速反應是套利實現的重要因素,這裡我們只講思路。那麼事情就簡單了:

- **對於事件驅動類**: 去市面上找一些滿足自己需求的即時總結 Web3 相關突發新聞的 BOT。
- **對於數據驅動類**: 會編程的自己寫寫(我自己還在學),不會編程的坐在電腦前面沒事看看漲幅跌幅榜或者去找提供這類數據的工具。

## 二、需要提前做好什麼準備?

### 1. 首先需要熟悉各個 Token 的主鏈

其實一共就 3 個類別:

- **EVM 系**: 指 Eth 網絡以及從 Eth 網絡延伸的 L2,包括但不限於 OP、BSC、Base、Avax 等等。
- **IBC 系**: 指 Atom 以及 Atom 延伸出的 Mantra、IRIS、Osmo 等等。
- **自有主網**: 指像 Sui、Sol、Near 這種獨立的主網。

### 2. 在掌握第 1 點的基礎上熟悉各種協議

#### (1) 找協議的功能性產品

比如 Defillama 和 Dexscreener 能幫你完整總結各個 Chain 以及 On-chain 協議,Coingecko 和 Coinmarketcap 能夠幫你總結和發現某個 Token 在鏈上是否具有流動性以及 Token 的漲跌幅,當然也有其他功能性產品可以自己去探索和發現。

#### (2) 有流動性的各種 DEX 和聚合器

尋找你的目標 Token 在鏈上的流動性主要分佈在哪些 DEX 協議上,沒有 TVL 極大的影響滑點和差價空間,很好理解。

- **EVM 系**: 每個鏈都有自己核心的 Swap,比如 ETH 網絡上的 Uniswap、BSC 的 Pancake、AVAX 網絡上的 LFJ、Abs 鏈上的 AbstractSwap 等等。當然,EVM 系已經比較成熟,現在都可用單鏈或者跨鏈聚合器來完成鏈上的 Swap,比如有大家熟知的 1inch、Paraswap、Matchaxyz、Kyberswap 等等。

- **IBC 系**: 流動性主要集中在 Osmo 這條 IBC 上,其他 IBC 也存在小眾的 Dex 可自行探索。

- **自有主網系**: Sol 上的 Jup、Sui 上的 Cetus 等等。

上述的羅列只是提供思路,其他協議大家都可以自行尋找和探索。

#### (3) 各種跨鏈橋和所需時間

這一點非常重要。你需要熟知市場上主流的跨鏈橋包括但不限於蟲洞 Wormhole、Axelar、deBridge 等等。

其實各個鏈也都會有他們自己官方認可的更具有針對性的跨鏈橋,這一點你也能從之前帖子內的 OM 的跨鏈上體會到。比如 Poly 網絡的 Pos 橋針對的是從 ETH 跨鏈到 Poly 網絡,OM 自己專屬的 mantra 橋,Sonic 自己專屬的官方橋等等。

一些針對自己主鏈的橋通常可以在官方網站的生態系統中找到,但是尤其要注意跨鏈的時間和跨鏈後的資產正確性。

**重要注意事項**:

- 有一些通過橋跨鏈到其他鏈的資產可能與目標鏈的資產不一致,例如你可以嘗試走蟲洞把 OM 跨鏈到 BSC 鏈上,你會發現完全是兩個東西。
- 還有一定就是跨進跨出的速度,這個需要犧牲一點手續費來作為嘗試,其他的跨鏈橋就不在這裡一一提及了。

## 三、出現價差空間如何操作?

### 1. 借助工具

通常情況下,當 CEX 的 Token 發生暴跌時,鏈上的反應速度會慢一些,就是很簡單從 CEX 提現到鏈上賣就行。為了鎖定利潤,你就必須要用到槓桿借貸或者合約套保這類工具。

現在的大部分情況下 CEX 的槓桿借貸額度很快會被用完(很多人用 BOT 借貸),所以可以繼續探索各個鏈上是否存在可用的借貸協議作為短期使用。

### 2. 底倉思路

在實例當中,會碰到鏈上和 CEX 都不存在可借貸的 Token 怎麼辦呢?這裡可以打開腦洞,運用粗暴的底倉思路。

這裡講一個我碰到的例子,之前 $COW 剛上線時就是在兩條不同的鏈上存在價差但是又沒有能借貸或者套保的地方。於是我就自己把 1K 個 Cow 當做我購買的底層,提前放在價格大概率發生變化的目標鏈上,當兩條鏈出現價差時,直接高價賣出底倉同時在價格低的一側買入。

這樣做的好處就是大大節省了跨鏈的時間,無非就是多了一點底倉的風險敞口,如果最終的利潤大於底倉的跌幅那就是 OK 的。不過這種思路僅限於能夠持續一段時間或者長期出現價差的 Token,有一定限制。

### 3. 學習和模仿

我們都知道,模仿是快速達到目標的方式。如果你發現自己並不能像別人一樣隨時發現可套利的空間,那就去觀察聰明錢是如何做的,這需要你能熟練運用鏈上的瀏覽器。

在發現一些一直在搬磚套利的鏈上地址後可以用 Debank 或者 Zerion 來追踪聰明錢的具體操作方式。追踪聰明錢和發現聰明錢地址的操作方法需要大家在鏈上探索,這裡不再過多贅述。

除此之外,歷史上存在很多次關於搬磚套利的大事件,很多優秀的選手也分享了他們對於這類事件的復盤,也可以在社媒上多去尋找。像不久前 JellyJelly 事件、$Luna 暴雷、$Boba 空投事件等等都可以作為典型案例來學習。
