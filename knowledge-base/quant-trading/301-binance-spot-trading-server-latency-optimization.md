## AWS 東京區域延遲比較

做幣安現貨量化服務時，如果對速度有要求，可以優先考慮將伺服器放在 AWS 東京的 **Zone C**。

### Ticker 速度排序

**Zone C > Zone D > Zone A**（針對永續合約）

### 延遲差異

- Zone C 比 Zone D 快約 **1.2ms**
- Zone C 比 Zone A 快約 **2.8ms**
