## 技術架構

### 核心特性

**零知識證明 Rollup (ZK Rollup)**
- 構建在以太坊之上，針對速度、吞吐量和規模進行優化
- 對所有操作（包括訂單匹配和清算）生成零知識證明

**高可擴展性 (SCALABLE)**
- 專為大規模設計，每秒可處理數萬筆訂單和取消操作
- 延遲時間僅為毫秒級

**高效率 (EFFICIENT)**
- 訂單匹配和正確性證明的基礎設施經過高度優化，成本低廉
- 散戶交易者零手續費
- 高頻交易者享有極具競爭力的費用

**可驗證性 (VERIFIABLE)**
- 嚴格遵循公開預定義的規則集
- 所有操作（包括匹配和清算）均通過密碼學證明，並在以太坊上公開驗證

**安全性 (SECURE)**
- 使用以太坊作為基礎層進行證明和系統狀態變更
- 用戶可以通過以太坊安全地存款或提款
- 所有證明均在公開環境下驗證

### 系統架構流程

```
INCOMING_TX → TX_QUEUE → MATCHING ENGINE → RISK ENGINE → EXECUTOR → BATCH PROVER → ETHEREUM
```

1. **交易流入 (INCOMING_TX)**: 交易通過自定義電路流轉，即時驗證簽名和餘額
2. **交易隊列 (TX_QUEUE)**: 每秒處理數千筆操作並生成加密證明，保持交易數據私密
3. **匹配引擎 (MATCHING ENGINE)**: 根據價格-時間優先級進行交易匹配，配合密碼學證明，消除審查或偏袒
4. **風險引擎 (RISK ENGINE)**: 監控保證金並自動清算高風險帳戶，由保險基金支持
5. **執行器 (EXECUTOR)**: 批量處理交易和清算，更新餘額並生成可驗證的證明
6. **批量證明器 (BATCH PROVER)**: 將多筆交易打包成一個緊湊的證明，一次驗證數百筆交易，提高速度並降低以太坊成本
7. **以太坊結算 (ETHEREUM)**: 最終在以太坊上結算，享有完整的區塊鏈安全性

**結果**: 達到交易所級別的速度，同時保持完全的鏈上透明度——每秒數千筆公平操作。
