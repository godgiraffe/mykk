## 核心概念：下架時差套利

Binance 宣布下架 ALPACA → 所有合約倉位必須按照指數價清算  
Gate 沒有宣布 → 市場還在自由交易，情緒驅動價格

一個「錨住了」，一個「飄起來了」→ 分叉 = 發散

如果都公告了，那麼 BN 和 Gate 將沒有價差，以穩定的指數價格結算。由於 BN 和 Gate 公告時間不一致，也就是一個會被結算，另一個在如此激情的市場下還在允許大伙交易的話，一定會出現巨大的價差。而假如當前價差只有 4%，還達不到巨大的水平的話，就一定意味著做「發散」套利是正確的。

## 案例：ALPACA 為什麼 Gate 和 Binance 價格相差 220%

> 引用 @Boywus

今天 16:00 的時候，Binance 的 ALPACA 開始爆拉，Gate 的 ALPACA 開始暴跌，差價一度來到了 220%，以及很多人問我為什麼可以預測價差會發散來盈利，開個貼解釋下。

### 發現異常價差

首先剛剛開始的時候，我看到 Gate 的價格遠遠低於 Binance 的價格 8% 的時候，還有 5% 資費吃，我做完一筆套利後發現這個事情還在持續，我就開始研究這個奇怪的情況。

### 關鍵資訊：下架公告差異

原來 Binance 和 Bybit 在 4/30 17:00 下架 Alpaca，所以他們之間的所有價差都是可以套利的，因為下架結算用的指數價格基本上是一致的。

但是 Gate 並沒有宣布下架，在確認這個消息後，我第一時間發了推特警告大家不要吃這個資費和做價差收斂套利。

### 價差發散策略

這個時候，從邏輯上來說已經確定可以做價差發散了，因為 Gate 會和 Binance 產生巨大價差，所以應該 Gate 繼續開空，Binance 開多。

但是這種操作基本上在幣圈幾年才出現一次，所以又發了一個帖子建議散戶不要做「價差發散」套利，對於小白來說是反直覺的，至於其他勇敢的人開進去，是市場對他們的獎勵。

### 風險提醒

想起 @sunlc_crypto 前幾天說的，套利行業對於很多小白來說，風險和收益不成正比；如果剛剛去做價差收斂可能兩個所的倉位就全爆倉了，至於 Pro 玩家做價差發散，他們的收益也是尊重市場規律，並且有一定的賭性才能拿到的。

提升自己，敬畏市場，錯過這次不可惜，下次我們一起努力。
