## 核心邏輯

### 為什麼可以低風險套利？

1. **時間窗口優勢**：黃金市場週末休市，XAUt0 價格幾乎不動
2. **集中流動性模式**：可設置極窄價格區間（±0.2% 以內）
3. **平台激勵**：Byreal 每日為 XAUt0-USDT 池投放數千美金獎勵
4. **無常損失極小**：窄區間 + 低波動 = 幾乎無無常損失

### 收益來源

- 交易手續費分潤（集中流動性放大收益）
- Byreal 平台激勵獎勵（主要收益來源）
- 借幣成本低於收益的利差

## 完整操作流程

### 步驟 1：在 Bybit 借幣

```
Bybit App → 質押借幣 → XAUT → 活期借幣
→ 輸入借幣數量 → 選擇抵押資產 → 借幣
```

**建議**：
- 計算好需要的 XAUT 數量
- 確認抵押率，避免爆倉風險
- 記錄借幣利率，確保套利空間

### 步驟 2：添加 XAUt0 代幣

```
OKX 錢包 App → 切換 SOL 網路 → 錢包首頁
→ 幣種管理 → 自定義幣種 → 合約地址欄
→ 貼上 XAUt0 合約地址 → 確認
```

**XAUt0 合約地址**：
```
AymATz4TCL9sWNEEV9Kvyz45CHVhDZ6kUgjTJPzLpU9P
```

### 步驟 3：提幣到 OKX 錢包

**接收地址準備**：
```
OKX 錢包首頁 → 接收 → 選擇 SOL 鏈的 XAUt0
→ 複製地址或截圖二維碼
```

**從 Bybit 提現**：
```
Bybit 資產頁面 → 提現 → 選擇 XAUT → 鏈上提幣
→ 填寫資訊 → 提現
```

| 欄位 | 填寫內容 |
|------|----------|
| 地址 | OKX 錢包的 SOL 地址 |
| 所屬網路 | SOL |
| 提現金額 | 借幣數量 |

**重要提醒**：
- 同樣步驟提現 USDT（等值或更多）
- 提現少量 SOL 作為 Gas 費

### 步驟 4：添加流動性

```
OKX 錢包 → 搜尋 Byreal 官網 → 右上角 Connect
→ 選擇 OKX Wallet
```

**設置 LP 參數**：
```
點擊 XAUt0-USDT 池子 → Add Liquidity
→ Set Price Range 區域拖動邊界線
→ 設置區間為 ±0.2% 以內
→ 下拉至頁面底部 → Create Position → 簽名確認
```

**關鍵設置**：
- 價格區間：**±0.2% 以內**（越窄越好，但需監控）
- 資金分配：根據當前價格自動計算 XAUt0/USDT 比例

### 步驟 5：管理持倉

**查看持倉**：
```
OKX 錢包 → Byreal 頁面 → Portfolio
→ 下拉 → Current Position
```

**領取獎勵**：
- 獎勵實時累積，可隨時 Claim

**撤出流動性**：
```
Byreal 頁面 → Create Position
→ 點擊 Share 旁的 ... 按鈕
→ Remove Liquidity → Amount 選擇 100%
→ Withdraw & Close Position → 簽名確認
```

### 步驟 6：歸還借幣

撤出流動性後：

1. 將 XAUt0 和 USDT 從 OKX 錢包充值回 Bybit
2. 在 Bybit 質押借幣頁面歸還 XAUT
3. 解除抵押資產
4. 計算實際收益（激勵獎勵 - 借幣利息 - Gas 費）

## 風險提示

### 主要風險

| 風險類型 | 說明 | 應對措施 |
|----------|------|----------|
| **價格偏離** | 黃金價格突破設置區間 | 監控持倉，及時調整區間 |
| **智能合約風險** | Byreal 協議漏洞 | 小額測試，分散資金 |
| **清算風險** | 借幣抵押率不足 | 保持充足抵押，監控抵押率 |
| **Gas 費波動** | Solana 網路擁堵 | 預留充足 SOL，避開高峰 |
| **APR 下降** | 參與人數增加稀釋收益 | 計算實時 APR，評估是否繼續 |

### 注意事項

1. **時間把控**：
   - 最佳進場：週五晚至週六早
   - 最佳出場：週日晚至週一早（黃金開盤前）

2. **資金規模**：
   - 首次操作建議小額測試
   - 計算借幣成本與預期收益的平衡點

3. **監控頻率**：
   - 雖然週末波動小，仍需每日檢查持倉狀態
   - 設置價格警報（如支援）

4. **合規性**：
   - 確認所在地區是否允許使用相關平台
   - 了解借幣和 DeFi 操作的稅務影響
