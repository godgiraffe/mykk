## 賽道現況

目前永續交易所（Perp DEX）賽道中，edgeX 和 lighter 已完成主要階段，正處於等待收穫期。

## 主攻項目分析

### 重點關注項目

| 項目 | 現況 | 投資評估 |
|------|------|----------|
| **Variational** | 剛啟動積分系統 | 具備早期 lighter 的特質，早期參與機會 |
| **Backpack** | 預計明年 Q1 發幣 | 1011 事件後大量用戶退出，反向操作可把握不卷的積分機會 |
| **Extended** | 前 Revolut 團隊打造 | 高度看好，具備早期 edgeX 的潛力 |
| **Paradex** | 明年 1 月底 TGE | 目前可轉帳 XP，等同變相提前空投，適合套利操作 |

### 觀察列表項目

| 項目 | 特點 | 待觀察事項 |
|------|------|-----------|
| **Ethereal** | Ethena 背景 | 疑似被 ENA 放棄，深度表現較差，需進一步考究 |
| **ApeX** | 國人團隊，已發幣 | 待進一步考察 |
| **Based** | Ethena 孵化 | Hype 中最強的 DEX，但手續費過高 |
| **grvt** | - | 團隊格局有待評估 |

## 投資策略

作者計劃深入研究上述項目，篩選優質標的後採取重倉策略。投資重點在於：

1. **早期項目**：Variational 和 Extended 具備早期紅利
2. **逆勢機會**：Backpack 在用戶流失期反向累積積分
3. **套利機會**：Paradex 的 XP 轉帳機制提供提前變現可能
4. **風險項目**：觀察列表項目存在團隊背景、深度、成本等隱憂，需謹慎評估
