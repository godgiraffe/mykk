## 概述

開源 Python 腳本用於 Euler/Lista 協議救援，已成功挽救 $26k 資金。腳本持續監測 Lista 金庫餘額，一旦檢測到資金即自動執行提款操作。

## 功能特性

### 已實現功能

- **Lista 金庫監測**：持續讀取 Lista 金庫餘額並自動提款
- **資金追蹤**：已成功救援 $26k 資金

### 開發中功能
