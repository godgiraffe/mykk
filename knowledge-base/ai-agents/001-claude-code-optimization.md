## Claude Code 協助優化技能：知識庫文章

### 總覽

本文探討 Claude Code 在技能優化方面的潛力。作者 @LotusDecoder 認為 Claude Code 的強大能力使其能夠加速技能的迭代和優化。文章重點在於：

*   **Claude Code 的能力：** 作者認為 Claude Code 的能力非常強大，如同坐上火箭般提升效率。
*   **技能迭代和優化：** 作者持續在迭代和優化 Skills。
*   **自主優化實驗：** 作者計畫保存 Claude Code 的會話紀錄，並嘗試讓它自主優化所有 Skills 和 Subagent。
*   **本地文件系統的優勢：** 基於本地文件系統的 Skill/Subagent 方便迭代，讓 Claude Code 協助維護更新，持續改進優化。

### Claude Code 的強大能力與技能迭代

@LotusDecoder 表示，Claude Code 的能力令人印象深刻，形容其效率提升如同「坐上火箭」。作者本身也一直在持續迭代和優化 Skills，但尚未嘗試完全交由 Claude Code 獨立優化改進。

### 利用會話紀錄進行自主優化

作者計畫保存全量的 Claude Code 會話紀錄，並設想可以透過一句簡單的指令，例如：

`ralph-loop：查閱歷史記錄並優化改進全部skills/subagent`

就能讓 Claude Code 自主進行優化。

### 本地文件系統的優勢與持續迭代

引用 @dotey 的觀點，基於本地文件系統的 Skill/Subagent 有個好處，就是可以持續迭代，讓 Claude Code 協助維護更新，持續改進優化，變得越來越好用。
