試用了一個新 AI IDE：Kiro，也是 VSCode 套皮，但發現了兩點比 Cursor 強的地方：

## UI 設計

UI 非常精緻，各種圓角很潤，並且和 VSCode 結合得很巧妙。比如 task 文件裡有角標來提示你當前的任務狀態。

## Prompt 調教

Prompt 調教得更好，對於複雜 task 會先生成 design、requirement、tasks 這三個 markdown 檔案來做開發計劃。
