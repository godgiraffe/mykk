## 外掛安裝步驟

1. **切換模型至 Opus**
   ```bash
   /model -> opus
   ```

2. **新增官方外掛市集**
   ```bash
   /plugin marketplace add anthropics/claude-code
   ```

3. **安裝 frontend-design 外掛**
   ```bash
   /plugin install frontend-design@claude-code-plugins
   ```

4. **啟用 Plan Mode**
   - 按 `shift+tab` 直到看到 plan mode

5. **開始使用**
   ```
   Use the frontend-design skill to [描述你想建立的介面]
   ```

## 背景

Opus 4.5 是目前在設計與視覺方面表現最佳的模型。搭配 Claude Code 的 frontend-design 外掛，可以快速產生高品質的使用者介面設計。
