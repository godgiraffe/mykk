## 專案結構

專案按工具分類整理，每個工具都有獨立資料夾，包含：
- **系統提示詞（System Prompts）**: 完整的 AI 指令設定
- **內部工具（Internal Tools）**: 工具內建的輔助功能
- **AI 模型配置（AI Models）**: 使用的模型與參數設定

主要分類：
```
├── Cursor Prompts/
├── Windsurf/
├── Claude Code/

├── Devin AI/

├── Google/
├── Open Source prompts/
└── ...（其他 20+ 工具）
```
