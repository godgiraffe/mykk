臥槽,剛發現一個 Claude Code 的增強工具!讓 Claude Code 對話完全可視化!

## 主要功能

📊 **2D 時間線視圖** - 對話歷史一目了然
🎯 **智能跳轉** - 快速定位優質 Prompt 學習
📁 **一键導出** - JSON/Markdown 格式都能支持

## 安裝方式

只需一行命令:

```bash
npm install -g @code-is-cheap/show-me-the-talk
```

## 特色

不僅代碼是開源的,連每個功能是怎麼指揮 AI 寫出來的 Prompt 也開源了,你也能只用 3 天就開發一個類似的 CLI 工具。
