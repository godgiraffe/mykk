## Claude Code 使用技巧分享

本篇文章整理自 @cryptonerdcn (NerdC) 對 Claude Code 創始人 Boris 的 Claude Code 使用經驗分享的解讀。重點在於如何讓 Claude Code Agent 避免重複犯錯，這是一個值得學習的技巧。

### 核心觀點：記錄錯誤與解決方案

Claude Code 團隊在使用 Claude Code 時的一個關鍵技巧是讓 Agent 記錄自身犯下的錯誤以及對應的解決方案。 具體做法是將這些資訊寫入一個文件（例如 `https://t.co/VyIzQQAUX2`），以便 Agent 在未來遇到類似問題時可以參考，避免重複犯錯。

### 技巧詳解：

*   **目的：** 避免 Claude Code Agent 重複犯錯。
*   **方法：** 讓 Agent 將錯誤和解決方案記錄在可存取的檔案中。
*   **優點：** 提高效率，減少不必要的重複嘗試。

### 其他注意事項：

*   Boris 強調，使用 Claude Code 沒有絕對正確的方法，每個人的設置和需求都不同。
*   建議使用者積極實驗，找到最適合自己的使用方式。
