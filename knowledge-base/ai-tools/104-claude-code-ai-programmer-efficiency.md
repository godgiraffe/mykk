## AI 程式員工作流程

### 完整開發流程

1. **需求定義**：使用 AI 撰寫 PRD（產品需求文件）
2. **任務切分**：透過 Task Manager 將專案拆分成具體任務
3. **設計階段**：使用 Claude Code 定義 3 個 UI 設計師
4. **環境隔離**：用 git worktree 切分工作環境，避免影響原有內容
5. **方案產出**：開始產出設計，三個方案中總能選出一個進行優化

### 精確調整工具

當大致方向確定後，安裝 Stagewise 插件可以針對性地選擇特定 component 進行精確性調整。

## 效率差距與技術紅利

運用好這些 AI 工具，在當下決定了普通程式員和 AI 程式員的效率差別可能是 **10 倍甚至百倍**。

或許在未來 2-3 年會被更傻瓜的工具抹平，但現在是有明顯的**技術紅利期**。
