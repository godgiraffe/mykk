## 各大模型訂閱清單

最近看到不少幣圈 KOL 在推 Gemini，這裡分享一下我自己的大模型使用清單與體驗：

| 模型/服務 | 訂閱方案 | 費用 |
|-----------|----------|------|
| Claude | Claude Pro | 20 USD/月 |
| GPT | GPT Plus | 20 USD/月 |
| Surf | 年費 Pro | 228 USD/年 |
| Gemini | 學生優惠薅到一年 Pro | 免費 |
| Perplexity | 年費會員（薅的） | 免費 |

## Gemini 使用建議

其實不薅 Gemini Pro 也行，直接用 **Google AI Studio**，完全免費，而且能最早體驗 Google 最新模型，比 Gemini App 更新更快，就是交互比 App 稍差。

## 最佳使用策略

根據不同場景選擇最適合的工具：

| 使用場景 | 推薦工具 |
|----------|----------|
| 搜索 + 寫文章 | Gemini / Google AI Studio |
| 文生圖 | Google Nano Banana Pro |
| 寫代碼 | Claude Code + Codex（哪個有額度用哪個，大型 Feature 用 Codex） |
| 通用問題 | GPT |
| Web3 分析找資料 | Perplexity / Surf |
