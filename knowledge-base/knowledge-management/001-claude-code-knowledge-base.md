## 利用 Claude Code 建立個人經驗知識庫

### 總覽

本文作者 @LotusDecoder 分享了如何使用 Claude Code 管理個人知識庫的經驗，核心概念是建立「經驗庫」，將反思、心得和道理記錄下來。當遇到類似問題時，可以向 Claude Code 提問，尋找經驗庫中相似的經歷作為參考，從而加速解決問題。

### 經驗庫的建立與管理

作者約在兩三個月前開始密集使用 Claude Code 進行個人知識庫管理。他認為最核心的用法是建立「經驗庫」。

具體做法如下：

*   **建立目錄：** 在 Claude Code 中新建一個 `experiences/` 目錄。
*   **記錄反思：** 將日常反思、內省，或從生活中獲得的道理、發現，輸入 Claude Code 並保存到 `experiences/` 目錄中。

### 經驗庫的應用

當遇到問題或需要執行大型專案時，可以透過以下方式向 Claude Code 提問，從經驗庫中尋找靈感：

*   **提問方式：** 向 Claude Code 提問：「我遇到了 XX 事情，請搜尋經驗庫，有哪些相同結構、類似的經歷，可供參考。」

### 使用 Claude Code 建立經驗知識庫的優點

*   **快速檢索：** 當遇到類似問題時，能快速從個人經驗中提取相關資訊。
*   **經驗積累：** 透過不斷記錄和反思，累積個人經驗，提升解決問題的能力。
*   **結構化知識：** 將經驗結構化地儲存在 Claude Code 中，方便日後查閱和利用。

### 總結

透過建立和善用經驗庫，能有效地管理個人知識，並在需要時快速提取相關經驗，提升工作效率和問題解決能力。 Claude Code 提供了一個便捷的平台來實現這個目標。
